## Using macOS to compile Locha Mesh Chat for iOS devices
Before to proceed you need to install some tools, the easiest way is installing [**Xcode** via the Mac App Store](https://itunes.apple.com/us/app/xcode/id497799835?mt=12), this also will install the iOS simulator and other stuffs.

If you have already installed **Xcode** on your computer, make sure it's version 9.4 or greater.
